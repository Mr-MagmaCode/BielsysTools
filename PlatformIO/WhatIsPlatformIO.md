# What is PlatformIO?

PlatformIO is an open-source ecosystem for IoT development that provides a unified platform for building, testing, and deploying embedded applications. It supports a wide range of microcontroller platforms and development boards, making it easier for developers to create and manage their projects.

PlatformIO offers a powerful and flexible development environment that integrates with popular code editors like Visual Studio Code. It provides features such as intelligent code completion, debugging tools, and a built-in library manager, which simplifies the development process and enhances productivity.

With PlatformIO, developers can easily manage their project dependencies, build and upload firmware to their devices, and collaborate with other developers. It also supports continuous integration and testing, allowing developers to automate their workflows and ensure the quality of their code.

Overall, PlatformIO is a valuable tool for embedded developers, providing a comprehensive and user-friendly environment for building and managing IoT projects.

For installation instructions for PlatformIO, please refer to the corresponding guide for the IDE you are using (Visual Studio Code and CLion): [Guid for different editors](../EditorGuides).
